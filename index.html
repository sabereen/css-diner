بازی CSS Diner

نگران نباشید، خوب جایی آمدید!

شما به این‌جا آمدید تا انتخابگرهای CSS را یاد بگیرید. انتخابگرها یا Selectorها مشخص می‌کنند که استایل‌های ما باید به کدام عنصرها اعمال شوند.

یک CSS Rule

p {
   margin-bottom: 12px;
}

در این مثال p انتخابگر است (همه‌ی عنصرهای <p> را انتخاب می‌کند) و استایلِ margin-bottom را به آن‌ها اعمال می‌کند.

نحوه‌ی بازی به این شکل است که باید در ویرایشگر زیر یک انتخابگر CSS بنویسید تا اشیاء خواسته شده را انتخاب کند، یعنی همان چیزهایی که درحال تکان خوردن هستند. اگر این کار را درست انجام دهید، به مرحله‌ی بعدی می‌روید.

موس را روی اشیاء روی میز ببرید تا کد HTML آن‌ها را مشاهده کنید.

این چیست؟
